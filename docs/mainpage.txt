\section Overview
VCCC is a cross-platform header-only library consisting of several modules written C++14. <br>
It provides various algorithms, implementing C++17 and C++20 feature in C++14.

\section compiler_support Compiler Support
- GCC 6.0 or above
- Clang 8.0 or above
- MSVC 15.0 or above
- Emscripten 2.0.14 or above (below versions are not tested)

\section Modules
- \subpage functional. Implementation of STL header [`<functional>`](https://en.cppreference.com/w/cpp/header/functional)
- \subpage literal. Python-like `_format` library
- \subpage log. Cross-platform logging library
- \subpage math. Mathematics
- \subpage numeric. Numerics
- \subpage optional. Implementation of STL header [`<optional>`](https://en.cppreference.com/w/cpp/header/optional)
- \subpage random. Random algorithms
- \subpage range. Python-like range class
- \subpage signal. Implementation of [Boost.Signals2](https://www.boost.org/doc/libs/1_83_0/doc/html/signals2.html)
- \subpage tuple. Implementation of STL header [`<tuple>`](https://en.cppreference.com/w/cpp/header/tuple)
- \subpage type_support. Syntax sugar for C++
- \subpage type_traits. Extra type_traits
- \subpage utility. Utilities

\section third_party_libraries Third Party Libraries
