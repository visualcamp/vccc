Overview

VCCC is a cross-platform header-only library consisting of several modules written C++14.
It provides various algorithms, implementing C++17 ... C++26 feature in C++14.

Compiler Support

  • GCC 6.0 or above
  • Clang 8.0 or above
  • MSVC 15.0 or above
  • Emscripten 3.1.57 or above (tested 2.0.14 ~)

Libraries

  • algorithm.
  • array.
  • Compare.
  • concepts. Concepts.
  • expected.
  • functional. Implementation of STL header <functional>
  • iterator. Iterators.
  • literal. Python-like _format library.
  • log. Cross-platform logging library.
  • math. Mathematics libraries.
  • memory.
  • numeric. Numerical algorithm libraries most functions support both iterator and variadics.
  • optional. Implementation of STL header <optional>
  • random. Random algorithms.
  • ranges. Ranges library.
  • signal. Implementation of Boost.Signals
  • span. vccc::span view
  • string_view. A lightweight object that offers read-only access to a string or a part of a string.
  • tuple. Implementation of STL header <tuple>
  • type support. Syntax sugar libraries.
  • type_traits. Implementation of STL header <type_traits> with some extra features.
  • utility. Utility libraries.
  • variant. Implementation of STL header <variant>
